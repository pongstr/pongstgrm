Readme

Pongstagr.am is a jQuery plugin that lets display your instagram media to your website using Bootstrap Front-end styles and modal-plugin.

Requirements:

For the most part, Instagramâ€™s API only requires the use of a client_id. A client_id simply associates your server, script, or program with a specific application. However, some requests require authentication - specifically requests made on behalf of a user. Authenticated requests require an access_token. These tokens are unique to a user and should be stored securely. Access tokens may expire at any time in the future more here.

  1. Instagram account, User ID and Access Token
  2. jQuery 1.8+
  3. bootstrap-modal.js (or bootstrap.js || bootstrap.min.js )

or you can head over here to use the plugin-only version (one that doesn't rely on Bootstrap Front-end).

usage:

  1. User ID: If you have zero idea what your user id is, you may head to this link.

  2. Access Token: to get your access token, you may head to this link, make sure you follow the instructions on the "How do I get my client id?" link.

  3. Stylesheets, jQuery Library, Bootstrap JS & the Plugin must be initialised before you call on the plugin. 

      <!-- Include stylesheets to header -->
  <link rel="stylesheet" href="/path/to/css/boostrap.min.css" />
  <link rel="stylesheet" href="/path/to/css/boostrap-responsive.min.css" />
  <link rel="stylesheet" href="/path/to/css/pongstagr.am.css" />

  <!-- Include Javascripts to footer -->
  <script href="//ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.min.js" /></script>
  <script href="/path/to/js/boostrap.min.js" /></script>
  <script href="/path/to/js/pongstagr.am.js" /></script>

    the call on the plugin like this: 

      <script>
    $(document).ready(function(){

        $('div#selector').pongstgrm({
          accessId    : YourAccessID,
          accessToken : YourAccessToken
        });
    
    });  
  </script>

    By default the plugin displays eight (8) most recent media.

Other Options

    show   : null, // string,  options: 'recent', 'feed', 'liked', 'user'
    count  : null, // integer, options: 1(min) - 40(max), instagram limits the maximum number of photos to 40
    pager  : null  // boolean, options:  true or false (enables/disable load more button)
